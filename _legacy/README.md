# _legacy

This folder contains archived code from the original PurpleSector prototype. It is kept for reference only and is not part of the current application.

- **PurpleSector/** — the original Streamlit web app (3-page UI built with Streamlit + Matplotlib)
- **\*.py** — standalone Matplotlib analysis scripts that predated the web app

The current application lives in `backend/` (FastAPI) and `frontend/` (React + Vite).
